Coldbox theme extension for easy and high revenue Google AdSense. It supports responsive, in-feed, in-article, matched content and auto ads!

== Description ==

== Installation ==

1. Upload `coldbox-ads-extension` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set your publisher ID and Slot IDs from the Theme Customizer
